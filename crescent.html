Crescent Darts

A modern web application for tracking darts games, born from finding an old dartboard in our apartment

Project Overview

What started as a fun way to keep score during roommate darts matches evolved into a full-featured web application. Crescent Darts combines modern web technologies with classic game rules to create an intuitive, real-time scoring system for both X01 and Cricket games.

Technical Implementation

Frontend Stack

  • Next.js 14 - App Router & RSC
  • TypeScript - Type safety
  • Tailwind CSS - Responsive design
  • Framer Motion - Smooth animations

Backend Infrastructure

  • Supabase - Database & storage
  • PostgreSQL - Data persistence
  • Real-time - Live game updates
  • Vercel - Deployment platform

Development Journey

What began as a simple score tracker transformed into a comprehensive gaming platform. The project showcases modern web development practices while solving a real need - making our darts games more engaging and competitive. The addition of features like player profiles, game history, and statistics tracking has turned casual games into an ongoing tournament among roommates.

Key Features

Game Types

  • X01 Games - 301, 501, 701 variants
  • Cricket - Traditional scoring rules
  • Double-Out Option - For tournament style
  • Statistics Tracking - Per player, per game

Player Features

  • Profile Photos - Camera integration
  • Stats History - Track improvement
  • Real-time Updates - Live scoring
  • Victory Celebrations - Animated wins
