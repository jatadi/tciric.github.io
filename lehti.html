Lehti

AI-Powered Personal Health Tracker - Turn your health data into actionable insights

Project Overview

Lehti is an intelligent health tracking application that helps users discover patterns in their symptoms and treatments through AI-powered insights. By leveraging machine learning algorithms, Lehti transforms raw health data into actionable knowledge, helping users understand their health patterns and make informed decisions about their wellbeing.

The application addresses the common challenge of managing personal health data by providing a comprehensive platform for symptom logging, treatment tracking, and pattern recognition. With its intuitive interface and powerful AI backend, Lehti makes health monitoring accessible and insightful for everyday users.

Screenshots

Dashboard Overview

Your personalized health insights at a glance - view your health stats, use quick action buttons to log symptoms or treatments, and follow the getting started guide for new users.

Symptom Tracking

Select the type and severity of your symptom, add notes, and set the time - everything gets reflected in the recent symptoms tab with timestamps and color-coded severity indicators.

Treatment Management

Choose from different treatment types (medication, supplement, exercise, therapy, etc.) with proper emojis, enter dosage and notes - all treatments appear in your history with visual type indicators.

AI-Powered Health Insights

View AI-generated health patterns with confidence levels, resolve alerts when you've acted on them, and filter between active and resolved insights to track your progress.

Key Features

Health Tracking

  • Symptom Logging - Track 13 different symptom types with severity ratings (1-10)
  • Treatment Management - Log medications, supplements, exercises, therapies
  • Rich Data Types - Notes, timestamps, dosages, and categories
  • Historical Tracking - Comprehensive timeline of health events

AI-Powered Insights

  • Pattern Recognition - Discover correlations between treatments and improvements
  • Predictive Alerts - Get notified when symptoms might spike
  • Treatment Effectiveness - See which interventions work best
  • Side Effect Detection - Identify potential adverse reactions

Smart Alerts System

The AI engine generates intelligent health insights such as:

  • Post-Treatment Analysis - "Headaches improve 2-4 hours after Ibuprofen"
  • Symptom Spikes - "Poor mood often precedes severe headaches"
  • Co-occurrence Patterns - "Walking correlates with reduced pain levels"
  • Resolvable Insights - Mark patterns as acknowledged and acted upon

Technical Implementation

Backend Stack

  • Laravel 11 - Modern PHP framework with elegant syntax
  • SQLite - Lightweight, serverless database
  • JWT Authentication - Secure token-based auth
  • Eloquent ORM - SQL injection protection

Frontend & Infrastructure

  • JavaScript & CSS - Responsive, interactive design
  • RESTful API - Clean API architecture
  • Railway/Render - Cloud deployment platform
  • Composer - PHP dependency management

Database Architecture

The application uses a well-structured relational database with four main tables:

  • Users - User accounts and authentication
  • Symptom Logs - Individual symptom entries with severity and notes
  • Treatments - Treatment/intervention records with types and dosages
  • Alerts - AI-generated health insights and patterns

Security Features

Lehti implements comprehensive security measures to protect sensitive health data:

  • JWT Authentication - Secure token-based authentication
  • Input Validation - Server-side validation for all inputs
  • SQL Injection Protection - Eloquent ORM parameterized queries
  • XSS Protection - Escaped output and CSP headers
  • Password Hashing - Bcrypt with configurable rounds

Development Process

The development of Lehti focused on creating an intuitive user experience while implementing sophisticated AI algorithms for pattern recognition. Key challenges included designing a flexible data model that could accommodate diverse health conditions, implementing real-time pattern analysis, and ensuring data privacy and security compliance for sensitive health information.

The AI component was carefully designed to provide meaningful insights while avoiding medical advice, focusing instead on helping users understand their own data patterns and trends.
