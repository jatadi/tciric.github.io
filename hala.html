Hala

A Letterboxd-inspired platform for sports fans to track and share their watching experience

Project Overview

Hala fills a unique gap in the social tracking space. While platforms like Letterboxd and Goodreads let users track films and books, there wasn't a cross-sport network for logging and sharing sports watching experiences. Starting with Formula 1, Hala provides fans with a platform to log races, share micro-reviews, and build shareable lists in a spoiler-safe environment.

Key Features

Core Functionality

  • Quick Match Logging - "Zap" races with a single tap
  • Rating & Reviews - 0-5 stars with micro-reviews
  • Personal Diary - Track your watching history
  • Spoiler-Safe Feed - Follow friends without spoilers

Social Features

  • Custom Lists - Create and share race collections
  • Follow System - Build your racing community
  • Real-time Updates - Instant feed updates
  • Community Focus - Share your F1 passion

Technical Implementation

Frontend Stack

  • Next.js 14 - App router & RSC for optimal performance
  • TypeScript - End-to-end type safety
  • tRPC - Type-safe API layer
  • Tailwind CSS - Modern, responsive design

Backend Infrastructure

  • Supabase - PostgreSQL database & Auth
  • Real-time Updates - Live feed synchronization
  • Row-Level Security - Robust data protection
  • Redis Cache - High-performance data access

Development Process

The development of Hala focused on creating a seamless user experience while ensuring robust performance and scalability. Key challenges included implementing real-time updates without compromising the spoiler-free nature of the platform, and designing a flexible data model that could later accommodate multiple sports beyond Formula 1.
