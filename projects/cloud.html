Cloud Bookshelf Application

Navigating a library's vast collection can be daunting for patrons and staff alike. The Bookshelf Management Application simplifies this process through a modern, cloud-based solution. Traditional library systems often struggle with integration and scalability, but this project overcomes such challenges using a microservices architecture.

The application is crafted with Java and Spring Boot, emphasizing clean and maintainable code, and is neatly containerized using Docker for easy deployment and scaling with Kubernetes. It features two primary microservices: the book-service for CRUD operations on books and the bookshelf-service for organizing collections, with RESTful APIs facilitating seamless communication between them.

Further enhancing user interaction, the application includes modules for searching, rating, and reviewing books, as well as managing bookshelves with ease. Each module is designed to be intuitive, providing a user-friendly interface that encourages engagement.

Additionally, the system's documentation and setup instructions are clear and concise, ensuring that anyone from tech enthusiasts to seasoned developers can deploy and utilize the application. The code and detailed documentation are readily available on GitHub for those interested in exploring or contributing to the project.
